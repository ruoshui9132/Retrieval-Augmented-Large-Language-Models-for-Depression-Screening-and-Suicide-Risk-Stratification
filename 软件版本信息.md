# 软件版本信息说明

## 一、运行环境

| 项目 | 版本要求 |
|------|----------|
| Python | >= 3.8 |
| 操作系统 | Windows / Linux / macOS |

## 二、核心依赖包

### 数据处理
| 包名 | 版本 | 用途 |
|------|------|------|
| pandas | >= 1.3.0 | 数据读取与处理 |
| numpy | >= 1.21.0 | 数值计算 |
| openpyxl | >= 3.0.0 | Excel文件读写 |

### 机器学习与统计
| 包名 | 版本 | 用途 |
|------|------|------|
| scikit-learn | >= 1.0.0 | 混淆矩阵、分类指标计算 |
| scipy | >= 1.7.0 | 统计分析、ICC计算 |

### 可视化
| 包名 | 版本 | 用途 |
|------|------|------|
| matplotlib | >= 3.4.0 | 混淆矩阵绑图 |
| seaborn | >= 0.11.0 | 热力图绑制 |

### 大模型API（对话生成）
| 包名 | 版本 | 用途 |
|------|------|------|
| openai | >= 1.0.0 | API调用 |
| requests | >= 2.26.0 | HTTP请求 |

## 三、使用的模型

| 模型 | 版本/来源 |
|------|----------|
| Baichuan-M2 | 百川智能API |
| DeepSeek-V3.1 | DeepSeek API |
| Qwen3-235B-A22B-Instruct-2507 | 阿里云通义千问API |
| GLM-4.6 | 智谱AI API |

## 四、安装说明

```bash
# 创建虚拟环境（推荐）
python -m venv venv
source venv/bin/activate  # Linux/macOS
# 或
venv\Scripts\activate  # Windows

# 安装依赖
pip install -r requirements.txt
```
